字体

加粗

需要加粗的文字左右用两个*号包裹

斜体

需要倾斜的文字左右用一个*号包裹

斜体加粗

需要倾斜与加粗的文字左右分别用三个*号包裹起来

删除线

需要加删除线则左右用~~包裹起来

引用

在引用的文字前加>即可。引用也可以嵌套，如加两个>>三个>>>
